# Fuzzy-cd - To fuzzy move into your directories

Finally a command that just works to cd with a TUI!

Works with both bash and sh.

## Requirements

First a Linux. Maybe MacOS too, I don't own any Apple device so I don't know.

Should be installed:

- fzf
- fd

Check the requirements with:
```bash
which fzf fd
```

This command should show two lines with paths to the binaries. If instead you see a `which: no ...`, then install what is not present.

## Compatibility

All POSIX-compatible shells, like: bash, sh, zsh, etc.

## How to install?

Run `install` with `sh ./install` and restart your terminal.
